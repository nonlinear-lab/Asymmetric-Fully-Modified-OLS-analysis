# Asymmetric-Fully-Modified-OLS-analysis-
this repository contains R-codes for the asymmetric fully modified OLS (AFMOLS) analysis. In other word, this analysis extends
the FMOLS to incoporate asymmetric decomposition (negative or positive) 
1. R-codes for the data-generating process, 1 white noise, 2 ARIMA (AR=0), 3 ARIMA (AR=0.8), 4 ARIMA (AR=0.95), 5. ARIMA (0,1,0) 6 Random walk
2. R-codes for the FMOLS analysis_cointRegFM function
3. R-codes for the FMOLS analysis_manual calculation
4. R-codes for the asymmetric FMOLS analysis_cointRegFM function
5. R-codes for the asymmetric FMOLS analysis_manual calculation
6. R-codes for Monte Carlo simulation for the asymmetric FMOLS analysis
7. R-codes for Monte Carlo simulation to compare seven different methods (OLS, DOLS, FMOLS, Fourier DOLS, Fourier FMOLS, Asymmetric DOLS, Asymmetric FMOLS_
8. R-codes for Monte Carlo simulation to compare seven different methods with four scenario (white noise, weak cointegration, very weak cointegration, and random walk)
It also includes seven different function files
Asymmetric DOLS function
Asymmetric FMOL function
DOLS function
FMOL function
Fourier DOLS function
Fourier FMOL function
simulation function
